# 17 – GSAP Showcase

A beginner-friendly animated page built with **HTML**, **CSS**, and **GSAP**.

## Animations Used

| Animation              | GSAP Feature              |
| ---------------------- | ------------------------- |
| Hero text fade in      | `gsap.to` + delay         |
| Cards reveal on scroll | `ScrollTrigger` + stagger |
| Button hover spring    | `elastic.out` easing      |
| Button click scroll    | `ScrollToPlugin`          |
| Footer fade in         | `ScrollTrigger`           |

## How to Run

```bash
npm run dev
```

Then open: `http://localhost:5173/pages/17_gsap-showcase/index.html`
